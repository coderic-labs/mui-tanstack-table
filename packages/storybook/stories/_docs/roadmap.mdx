# Roadmap

This project is actively improving the MUI rendering layer for TanStack Table. Planned work includes:

## Near-term priorities

- Improve documentation with full end-to-end examples
- Add more tutorial-style Getting Started content
- Add sample integrations for server-side filtering and pagination
- Add new toolbar and filter UI helper components

## Planned improvements

- Better support for virtualization-ready table layouts
- Better global filter and search components
- More built-in row expansion and detail rendering helpers
- Enhanced accessibility support for complex table interactions

## Goals

- Keep the API modular and composable
- Make it easy to build data-rich MUI tables with TanStack Table state
- Avoid coupling UI controls to table behavior
- Support gradual adoption in existing React apps
