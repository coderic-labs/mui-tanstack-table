# MUI TanStack Table

> A modular renderer for TanStack Table using Material UI.

This library provides a lightweight, render-only MUI layer for `@tanstack/react-table`. It lets you compose advanced table UIs using reusable MUI components while keeping all table state and logic inside TanStack Table.

## Start here

- **Getting Started** — Learn how to configure TanStack Table and render it with MUI components.
- **About** — Why this library exists and how it is designed.
- **APIs** — Detailed reference for architecture, exports, filtering, sorting, and paging contracts.
- **Roadmap** — What features are planned next.
- **Changelog** — Current release notes and version history.

## Why use this package?

- **Render-only UI** — UI components consume a configured table instance, they do not contain table logic.
- **Modular components** — Use only the toolbar, filtering, pagination, selection, expansion, column visibility, or column order (reordering) features you need.
- **MUI styling** — Built for Material UI so tables fit into MUI-based applications.
- **TanStack Table compatibility** — Works with TanStack Table v8 logic.
