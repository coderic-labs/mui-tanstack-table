# MUI TanStack Table

> A modular renderer for TanStack Table using Material UI.

This library provides a lightweight, render-only MUI layer for `@tanstack/react-table`. It lets you compose advanced table UIs using reusable MUI components while keeping all table state and logic inside TanStack Table.
