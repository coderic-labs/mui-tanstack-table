# Getting Started

This guide shows how to go from a TanStack Table configuration to a rendered MUI table using the library’s reusable components.

## 1. Install the package

```bash
npm install @coderic-labs/mui-tanstack-table
```

### Peer dependencies (supported ranges)

Install compatible peer dependencies in your app. The current supported ranges are:

- `react`: `>=18 <20`
- `react-dom`: `>=18 <20`
- `@mui/material`: `>=6 <8`
- `@mui/icons-material`: `>=6 <8`
- `@mui/x-date-pickers`: `>=7 <8`
- `@tanstack/react-table`: `>=8 <9`
- `@emotion/react`: `>=11 <12`
- `@emotion/styled`: `>=11 <12`
- `react-intl`: `>=7 <8`

## 2. Configure TanStack Table

Create your column definitions and table instance with `useReactTable`.

```tsx
import * as MTT from '@coderic-labs/mui-tanstack-table';
import { createColumnHelper, useReactTable, getCoreRowModel, getFilteredRowModel, getPaginationRowModel, getSortedRowModel } from '@tanstack/react-table';
import { Paper, Stack, TableContainer } from '@mui/material';

const columnHelper = createColumnHelper();

const columns = [
  columnHelper.accessor('name', {
    header: MTT.TableHeader,
    filter: MTT.predefinedColumnFilters.TextFilter,
    filterFn: 'includesString',
  }),
  columnHelper.accessor('age', {
    header: MTT.TableHeader,
    sortingFn: 'alphanumeric',
  }),
  columnHelper.accessor('status', {
    header: MTT.TableHeader,
    filter: MTT.predefinedColumnFilters.SelectFilter,
    filterFn: 'equals',
  }),
];

const table = useReactTable({
  data: data,
  columns,
  getCoreRowModel: getCoreRowModel(),
  getFilteredRowModel: getFilteredRowModel(),
  getPaginationRowModel: getPaginationRowModel(),
  getSortedRowModel: getSortedRowModel(),
});
```

## 3. Render the table with MUI components

Use `MTT.Table` and optional toolbar components from the package.

```tsx
return (
  <Stack gap={2}>
    <MTT.TableToolbar>
      <MTT.TableToolbarInfo>
        <MTT.TableResultsLabel table={table} />
      </MTT.TableToolbarInfo>
      <MTT.TableToolbarActions>
        <MTT.TableColumnVisibilityToggle table={table} />
      </MTT.TableToolbarActions>
    </MTT.TableToolbar>

    <TableContainer component={Paper}>
      <MTT.Table table={table} />
    </TableContainer>

    <MTT.TablePaginationV2 table={table} />
  </Stack>
);
```
