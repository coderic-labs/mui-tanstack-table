# TableRowSelectionHeader

## Usage description

Render a select-all checkbox in a display column header.

## Shape

```ts
TableRowSelectionHeader<T>({ table }: HeaderContext<T, unknown>)
```

## Props

### Type definition

```ts
type TableRowSelectionHeaderProps<T> = HeaderContext<T, unknown>;
```

### Own props

No custom props are defined by this library; this component consumes TanStack header context.
