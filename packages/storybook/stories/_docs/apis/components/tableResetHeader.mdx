# TableResetHeader

## Usage description

Render a utility header button that resets active table state.

## Shape

```ts
TableResetHeader<T>({ table }: HeaderContext<T, unknown>)
```

## Props

### Type definition

```ts
type TableResetHeaderProps<T> = HeaderContext<T, unknown>;
```

### Own props

No custom props are defined by this library; this component consumes TanStack header context.
