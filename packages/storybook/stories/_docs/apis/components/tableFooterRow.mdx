# TableFooterRow

## Usage description

Render a single footer group row within the table footer. Used internally by `Table` and available for composing custom table layouts.

## Shape

```ts
TableFooterRow<T>(props: TableFooterRowProps<T>)
```

## Props

### Type definition

```ts
type TableFooterRowProps<T> = {
  footerGroup: HeaderGroup<T>;
  stickyFooter?: boolean;
  tableLayout: 'auto' | 'fixed';
  footerGroupIndex: number;
};
```

### Own props

- `footerGroup`
  Type: `HeaderGroup<T>`
  Required TanStack header group used to render footer cells.
- `stickyFooter`
  Type: `boolean | undefined`
  Optional flag to keep footer cells sticky at the bottom.
- `tableLayout`
  Type: `'auto' | 'fixed'`
  Controls whether a filler cell is appended in `fixed` layout mode.
- `footerGroupIndex`
  Type: `number`
  Zero-based index of the footer group, used for test attribute assignment.
