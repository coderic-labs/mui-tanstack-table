# TableLocalizationProvider

## Usage description

Wrap table UI to provide locale and message overrides for built-in labels.

## Shape

```ts
TableLocalizationProvider(props: LocalizationProviderProps)
```

## Props

### Type definition

```ts
type LocalizationProviderProps = {
  locale: string;
  messages: Partial<typeof enMessages>;
  children: React.ReactNode;
};
```

### Own props

- `locale`
  Type: `string`  
  Locale code passed to `react-intl`.
- `messages`
  Type: `Partial<typeof enMessages>`  
  Partial translation map merged with default English messages.
- `children`
  Type: `React.ReactNode`  
  Descendant tree that consumes localization context.
