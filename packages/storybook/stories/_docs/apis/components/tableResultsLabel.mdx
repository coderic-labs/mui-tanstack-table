# TableResultsLabel

## Usage description

Show localized results summary based on table state.

## Shape

```ts
TableResultsLabel<T>(props: TableResultsLabelProps<T>)
```

## Props

### Type definition

```ts
type TableResultsLabelProps<T> = Omit<TypographyProps, 'children'> & {
  table: Table<T>;
};
```

### Own props

- `table`
  Type: `Table<T>`  
  Required TanStack table used to compute displayed and selected row counts.
