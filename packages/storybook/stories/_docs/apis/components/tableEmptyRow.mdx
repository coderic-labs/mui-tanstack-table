# TableEmptyRow

## Usage description

Render a full-width empty state row when the table has no data rows. Displays a localized "no results" message centered in the cell.

## Shape

```ts
TableEmptyRow<T>(props: TableEmptyRowProps<T>)
```

## Props

### Type definition

```ts
type TableEmptyRowProps<T> = {
  table: Table<T>;
  tableLayout: 'fixed' | 'auto';
};
```

### Own props

- `table`
  Type: `Table<T>`
  Required TanStack table instance used to compute colspan across visible leaf columns.
- `tableLayout`
  Type: `'fixed' | 'auto'`
  Controls colspan calculation — adds one extra column in `fixed` layout to cover the filler cell.
