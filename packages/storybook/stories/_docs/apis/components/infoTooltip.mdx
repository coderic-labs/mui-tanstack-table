# InfoTooltip

## Usage description

Render contextual help text with a standard info icon anchor.

## Shape

```ts
InfoTooltip(props: InfoTooltipProps)
```

## Props

### Type definition

```ts
type InfoTooltipProps = Omit<TooltipProps, 'children'> & {
  AnchorProps?: SvgIconProps;
};
```

### Own props

- `AnchorProps`
  Type: `SvgIconProps | undefined`  
  Optional props forwarded to the info icon anchor.
