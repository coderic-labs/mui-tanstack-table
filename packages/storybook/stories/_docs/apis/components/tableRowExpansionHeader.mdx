# TableRowExpansionHeader

## Usage description

Render a header-level control to collapse expanded rows.

## Shape

```ts
TableRowExpansionHeader<T>({ table }: HeaderContext<T, unknown>)
```

## Props

### Type definition

```ts
type TableRowExpansionHeaderProps<T> = HeaderContext<T, unknown>;
```

### Own props

No custom props are defined by this library; this component consumes TanStack header context.
