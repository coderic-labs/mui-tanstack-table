# TablePaginationV2

## Usage description

Render MUI TablePagination integrated with TanStack state.

## Shape

```ts
TablePaginationV2<TData>(props: TablePaginationV2Props<TData>)
```

## Props

### Type definition

```ts
type TablePaginationV2Props<TData> = Omit<TablePaginationProps, 'count' | 'rowsPerPage' | 'page' | 'onPageChange' | 'onRowsPerPageChange'> & {
  table: Table<TData>;
};
```

### Own props

- `table`
  Type: `Table<TData>`  
  Required TanStack table used to bind MUI TablePagination state.
