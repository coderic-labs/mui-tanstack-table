# TableHeader

## Usage description

Use as a column header renderer for inline sorting and filtering interactions.

## Shape

```ts
TableHeader<TData, TValue>(context: HeaderContext<TData, TValue> & StackProps)
```

## Props

### Type definition

```ts
type TableHeaderProps<TData, TValue> = HeaderContext<TData, TValue> & StackProps;
```

### Own props

No custom props are defined by this library; this component consumes TanStack header context (plus Stack props).
