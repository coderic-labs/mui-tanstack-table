# TableFilterOverview

## Usage description

Display active filters and provide quick filter-reset controls.

## Shape

```ts
TableFilterOverview<T>(props: TableFilterOverviewProps<T>)
```

## Props

### Type definition

```ts
type TableFilterOverviewProps<T> = Omit<TypographyProps, 'children'> & {
  table: Table<T>;
  formatFilterValue?: (filter: ColumnFilter) => string;
};
```

### Own props

- `table`
  Type: `Table<T>`  
  Required TanStack table used to read and reset active column filters.
- `formatFilterValue`
  Type: `((filter: ColumnFilter) => string) | undefined`  
  Optional formatter for displayed filter chip values.
