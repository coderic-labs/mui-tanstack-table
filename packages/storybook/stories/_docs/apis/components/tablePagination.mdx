# TablePagination

## Usage description

Render stack-based pagination controls bound to TanStack pagination state.

## Shape

```ts
TablePagination<TData>(props: TablePaginationProps<TData>)
```

## Props

### Type definition

```ts
type TablePaginationProps<TData> = StackProps & {
  table: Table<TData>;
  options?: number[];
};
```

### Own props

- `table`
  Type: `Table<TData>`  
  Required TanStack table with pagination state and handlers.
- `options`
  Type: `number[] | undefined`  
  Optional rows-per-page options.
