# TableRowExpansionCell

## Usage description

Render a row-level expand/collapse control.

## Shape

```ts
TableRowExpansionCell<T>({ row }: CellContext<T, unknown>)
```

## Props

### Type definition

```ts
type TableRowExpansionCellProps<T> = CellContext<T, unknown>;
```

### Own props

No custom props are defined by this library; this component consumes TanStack cell context.
