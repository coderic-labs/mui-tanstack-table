# Column Pinning

Column pinning keeps selected columns visible while scrolling horizontally.

This package currently supports a simple setup: pin the first column to the left and the last column to the right.

## Usage notes

- Keep pinning simple: first column left, last column right.
- More advanced pinning flows are intentionally out of scope for this package.
- Use a scrollable table container when needed for smaller viewports.

> Note: this functionality is temporary and will be replaced by a more advanced column pinning mechanism that supports pinning multiple columns.
