# Row Selection API

Row selection uses the same TanStack table state in client-side and server-side tables. The selection column renderers and bulk-action helpers are shared, so you do not need separate integration variants.

## Shared approach

- Enable `enableRowSelection: true`.
- Provide a stable `getRowId` so selected rows can be tracked reliably.
- Render `TableRowSelectionHeader` in the selection column header.
- Render `TableRowSelectionCell` in the selection column body.
- Read selected ids from `table.getState().rowSelection` directly or use `TableBulkActionButton`.
