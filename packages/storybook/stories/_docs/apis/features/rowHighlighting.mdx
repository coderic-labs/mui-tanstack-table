# Row Highlighting

The `Table` component accepts a `getRowStyle` prop that lets you apply conditional styles to individual cells. Use it to highlight entire rows by returning the same style for all cells in a row based on `cell.row.original` data.
By setting variable `--rowcolor` in the returned style, you can control the background color of the entire row. Cells in this row will use 'var(--rowcolor)' as their background color.

## Type

```ts
type GetCellStyle<T> = (cell: Cell<T, unknown>) => StyleValue | undefined;
```

Where `StyleValue` is an MUI `sx` style object: `SystemStyleObject<Theme> | ((theme: Theme) => SystemStyleObject<Theme>)`.

## Usage

Pass a `getRowStyle` function to `<Table>`:

```tsx
const getRowStyle = (row: Row<Developer>) => {
  if (row.original.employmentType === EmploymentType.intern) {
    return { '--rowcolor': 'warning.light' };
  }
};

<Table table={table} getRowStyle={getRowStyle} />
```

- Return a style object to apply it, or `undefined` to leave the cell unstyled.
- Use `row.original` to access row data for conditional logic.
- Use MUI theme tokens (`warning.light`, `error.main`, etc.) to stay consistent with the theme.
- Use `--rowcolor` variable to set the background colors for all cells in a row.
