# Sticky Header

Use the `stickyHeader` prop on `Table` to keep header cells fixed at the top while table rows scroll.

## Usage

```tsx
<TableContainer sx={{ maxHeight: 360 }}>
  <Table table={table} stickyHeader />
</TableContainer>
```

- Set `stickyHeader` on `Table`.
- Put the table in a scrollable container (`TableContainer` with `maxHeight` or fixed height).
- Keep your column headers as usual (`TableHeader`, `TableHeaderV2`, etc.).
