# Table Extension

The package augments TanStack `ColumnDefBase` with UI-focused metadata keys.

## Extended column keys

- `filter`: header filter renderer.
- `tooltip`: tooltip renderer/content for header context.
- `title`: optional display title used by header components.

## Module augmentation contract

```ts
interface ColumnDefBase<TData extends RowData, TValue = unknown> {
  filter?: ColumnDefTemplate<HeaderContext<TData, TValue>>;
  tooltip?: ColumnDefTemplate<HeaderContext<TData, TValue>>;
  title?: string;
}
```

## Why this exists

The extension keeps TanStack in charge of table logic while letting columns carry rendering hints used by MUI header/filter UI.
